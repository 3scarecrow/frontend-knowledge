# 元素隐藏

  • visibility: hidden 使元素不可见，但仍占据页面空间
  • opacity: 0 设置元素为完成透明，但仍占据页面空间
  • display: none 不显示元素并不占据页面空间
  • position: absolute 绝对定位使元素定位到可视区之外
  • transform: scale(0) 将元素缩放至 0

# 区别

--- display:none visibility: hidden opacity: 0

是否回流和重绘 回流与重绘 重绘 重绘
是否触发事件 否 否 是
